### Website 🌐
This is my professional website, coded by hand and deployed through an experimental Rube-Goldberg machine of static site generators and GitHub actions. If you've stumbled upon this and wish to look through the repository, I cannot promise the most concise way of using GitHub Pages or cleanest coding practices therein. However, it appears to work... which is good enough for a fun side project of mine!

---
#### Major Resources ✍️
- [11ty](https://www.11ty.dev/) (Static Site Generator)
- [Nunjucks](https://mozilla.github.io/nunjucks/) (Templating Language)
- [Bootstrap 5.0](https://getbootstrap.com/) (CSS Template)
- [Deploy to GitHub Pages](https://github.com/peaceiris/actions-gh-pages) (Publishing Action)
