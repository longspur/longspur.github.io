### Credits

This site was proudly coded from the ground-up with my own two hands, without the use of generative AI. It may seem simpler to ask a program to generate a layout, or to use a WYSIWYG editor (drag-and-drop), but I enjoy the challenge of understanding the bones that make up my site. Additionally, I have much more control over the layout. So while it takes more time, I am much more satisfied with the result. 

Nonetheless, the code did not appear from the ether! Here are some of the resources I used to construct my site.

---
- [**11ty**](https://www.11ty.dev/) and [**Nunjucks**](https://mozilla.github.io/nunjucks/) - My static site generator and templating language, which allows me to code and apply the base "layout" of the site to all pages.
- [**Deploy to GitHub Pages**](https://github.com/peaceiris/actions-gh-pages) - GitHub action that deploys the final site to be viewed publicly.
- [**Bootstrap Documentation**](https://getbootstrap.com/docs/4.0/getting-started/introduction/) - This is the CSS framework I used for this site, especially helpful for setting up the navbar and footer.
- [**FSLightbox**](https://fslightbox.com/javascript) - The "lightbox" effect script that makes certain images larger on a click.
- [**Theme Switcher**](https://albertoroura.com/building-a-theme-switcher-for-bootstrap/) - Viewing mode toggle for Light/Night/System preference.
- [**FontAwesome**](https://fontawesome.com/) - Icons used around the site.
